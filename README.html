C2C Server App

Peer-to-Peer Communication in Action

Introduction

Welcome to the C2C Server App! This project enables peer-to-peer (P2P) communication by allowing clients to connect directly with one another, forming a decentralized network. It is built to support real-time messaging, file transfers, and more.

Features

  • Real-time messaging between clients
  • File transfer capabilities
  • End-to-end encryption for secure communication
  • Lightweight and scalable server architecture
  • Easy to integrate with existing systems

Installation

Follow these steps to get the C2C Server App running on your local machine:

$ git clone https://github.com/your-username/c2c-server-app.git
$ cd c2c-server-app
$ npm install
$ npm start

Make sure you have Node.js installed. You can download it from here.

Usage

Once the app is up and running, follow these steps to connect to other peers:

  • Run the server with npm start.
  • Open the app in your browser (usually at http://localhost:3000).
  • Each client will be given a unique identifier (peer ID) for connecting with others.
  • Use the interface to initiate connections with other peers, send messages, and transfer files.

Enjoy seamless client-to-client communication!

Technologies Used

  • Node.js - For server-side operations
  • WebSocket - For real-time communication
  • Express.js - Web framework for building the server
  • Socket.io - For bidirectional communication between clients and server
  • PeerJS - To handle peer-to-peer connections
  • JavaScript (ES6+) - For client-side scripting
